Software & Data

You can find here several of the code I developed these years for teaching and research:

  • PEPINO: Packer (with) Educational Purposes (for) bINary Obfuscation
    It is an educational packer developed by Xabier Ugarte-Pedrero that implements a Shifting Decoding Frames approach to avoid memory dumping.
    • Download the original source code
    • Download the packed version
  • CompressionTextClassifier
    Text classification plugin for Weka 3.7. It implements a text classifier based on Variable Markov Models for compression.
    • Download (version 0.4.3).
